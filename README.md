# raw-react

## Features

- [x] 支持`React 17.x`
- [x] 支持`less`
- [x] 支持`webpack5`
- [x] 支持typescript
- [x] 使用`babel`编译`typescript`
- [x] 支持静态文件导入
- [x] 支持HMR[React Hot Loader](https://github.com/gaearon/react-hot-loader)
- [x] 支持prettier限制代码风格
- [x] 支持eslint代码规范
- [x] 支持commit规范限制
- [ ] 支持路由
- [ ] 支持Tree Shaking
